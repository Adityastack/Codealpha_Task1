# ECommerce Store

A basic e-commerce web application developed using HTML, CSS, and JavaScript.

## Features

* Product listing page
* Product details section
* Shopping cart functionality
* User registration and login
* Order processing and checkout
* Local storage data persistence
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage

## Project Description

This project simulates a simple online shopping platform where users can browse products, add items to their cart, register/login, and place orders. The application stores user, cart, and order information using browser local storage to demonstrate e-commerce functionality without requiring a backend database.

## How to Run

1. Download the project files.
2. Open `index.html` in a browser or run it using VS Code Live Server.
3. Browse products and test the shopping features.
